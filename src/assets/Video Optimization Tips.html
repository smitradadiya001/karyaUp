If you just drop an .mp4 into your hero section, it will slow things down unless you handle it properly. The trick is: optimize + lazy-load + show a fallback instantly.

Here’s how to do it the right way 👇

🔥 1. Compress your video first (MOST IMPORTANT)

Your MP4 should be lightweight.

  • Use tools like:

    • HandBrake

    • FFmpeg

👉 Settings:

  • Resolution: 720p (not 1080p unless needed)

  • Format: H.264

  • Bitrate: ~1000–2500 kbps

⚡ 2. Use optimized <video> tag (best practice)

HTML
<video
  autoPlay
  muted
  loop
  playsInline
  preload="metadata"
  poster="/images/hero-fallback.jpg"
  class="hero-video"
>
  <source src="/videos/hero-video.mp4" type="video/mp4" />
</video>

Important things here:

  • preload="metadata" → loads fast (not full video)

  • poster → shows image instantly (no blank screen)

  • muted → required for autoplay in browsers

  • playsInline → works on mobile

⚡ 3. Show image first, then load video (BEST UX)

Use a fallback image so users don’t feel loading:

HTML
<div class="hero">
  <img src="/images/hero-fallback.jpg" class="hero-img" />

  <video
    autoPlay
    muted
    loop
    playsInline
    preload="none"
    class="hero-video"
  >
    <source src="/videos/hero-video.mp4" type="video/mp4" />
  </video>
</div>
CSS
.hero {
  position: relative;
}

.hero-img,
.hero-video {
  position: absolute;
  width: 100%;
  height: 100%;
  object-fit: cover;
}

👉 Then use JS to load video after page load:

JavaScript
window.addEventListener("load", () => {
  const video = document.querySelector(".hero-video");
  video.load();
});

🚀 4. Use CDN (BIG performance boost)

Host video on:

  • Cloudflare

  • Amazon S3

👉 This reduces load time globally.

⚡ 5. Advanced (Best of best)

If you want zero lag feeling:

  • Convert video to .webm also

  • Use both formats:

HTML
<video autoPlay muted loop playsInline preload="metadata">
  <source src="/videos/hero-video.webm" type="video/webm" />
  <source src="/videos/hero-video.mp4" type="video/mp4" />
</video>

🧠 Key Tips (don’t skip)

  • Keep video under 2–3 MB

  • Keep duration 5–10 sec loop

  • Avoid audio

  • Avoid huge resolution

✅ Final mindset

👉 Don’t think “video = background”
👉 Think “video = lightweight animation”
